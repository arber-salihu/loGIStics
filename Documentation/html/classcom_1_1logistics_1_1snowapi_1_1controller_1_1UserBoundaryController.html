Detailed Description

REST controller for managing UserBoundary entities.

This controller provides endpoints for various CRUD operations related to user-boundary associations, enabling the retrieval, creation, and deletion of these associations via HTTP requests.

The controller is mapped to the base URL '/api/userBoundaries', from which all its endpoints are derived.

Methods:

  • getAll() - Retrieves a list of all user-boundary associations.
  • getById(Long, Long) - Retrieves a specific user-boundary association by composite key.
  • create(UserBoundary) - Creates a new user-boundary association.
  • delete(Long, Long) - Deletes a user-boundary association by its composite key.
