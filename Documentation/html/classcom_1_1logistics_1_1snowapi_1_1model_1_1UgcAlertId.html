Detailed Description

Embeddable class representing the composite key for the UgcAlert entity.

This class defines the fields that make up the primary key for the UgcAlert table, including a UGC code and an alert ID. It implements java.io.Serializable to ensure that instances of this class can be serialized by JPA and other Java technologies.

The fields:

  • ugcCode
    - Represents the UGC (Universal Geographic Code), a unique identifier for a geographic zone.
  • alertId
    - Represents the identifier for an Alert, linking the alert to a specific UGC zone.
