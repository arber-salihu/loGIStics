Detailed Description

REST controller for managing SnowUser entities.

This controller provides endpoints for various CRUD operations related to users, enabling the retrieval, creation, update, and deletion of user data via HTTP requests.

The controller is mapped to the base URL '/api/users', from which all its endpoints are derived.

Methods:

  • getAllUsers() - Retrieves a list of all users.
  • getUserById(Integer) - Retrieves a specific user by their ID.
  • createUser(SnowUser) - Creates a new user.
  • updateUser(Integer, SnowUser) - Updates an existing user identified by ID.
  • deleteUser(Integer) - Deletes a user by their ID.
