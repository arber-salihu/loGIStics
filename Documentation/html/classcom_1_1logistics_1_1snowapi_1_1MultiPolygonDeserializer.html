Detailed Description

Custom deserializer for converting JSON data representing a multi-polygon into a MultiPolygon object.

This deserializer handles the JSON structure specific to multi-polygon geometries, ensuring that they are correctly parsed into JTS's MultiPolygon format.

The JSON data must contain a "type" field with the value "MultiPolygon" and a "coordinates" field that lists the sets of coordinate arrays for each polygon. Each polygon must define an outer boundary and can optionally define one or more inner boundaries (holes).

Usage: This deserializer is typically used in conjunction with Jackson annotations to directly map JSON data onto MultiPolygon fields in model classes.

Example: 

@JsonDeserialize(using = MultiPolygonDeserializer.class)


private MultiPolygon theGeom;
