Detailed Description

Embeddable class representing the composite key for the UserBoundary entity.

This class defines the fields that make up the primary key for the user_boundary table, specifically the user ID and boundary ID. It implements java.io.Serializable to ensure that instances of this class can be serialized by JPA and other Java technologies.

Fields:

  • userId
    - The identifier for a SnowUser, part of the composite key.
  • boundaryId
    - The identifier for a Boundary, part of the composite key.
