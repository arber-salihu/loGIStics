Detailed Description

REST controller for managing Alert entities.

This controller provides endpoints for various CRUD operations related to alerts, making it possible to retrieve, create, update, and delete alerts through HTTP requests.

The controller is mapped to the base URL '/api/alerts', from which all its endpoints are derived.

Methods:

  • getAllAlerts() - Retrieves a list of all alerts.
  • getAlertById(Integer) - Retrieves a specific alert by its ID.
  • createAlert(Alert) - Creates a new alert.
  • updateAlert(Integer, Alert) - Updates an existing alert identified by ID.
  • deleteAlert(Integer) - Deletes an alert by its ID.
