Detailed Description

Embeddable class representing the composite key for the UgcBoundary entity.

This class defines the fields that make up the primary key for the UgcBoundary table, including a UGC (Universal Geographic Code) and a boundary ID. It implements java.io.Serializable to ensure that instances of this class can be serialized by JPA and other Java technologies.

Fields:

  • ugcCode
    - The UGC code, a unique identifier for a geographic zone, serving as part of the composite key.
  • boundaryId
    - The identifier for a Boundary, linking the boundary to a specific UGC zone as part of the composite key.
