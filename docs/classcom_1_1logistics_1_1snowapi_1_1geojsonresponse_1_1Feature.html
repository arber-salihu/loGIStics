Detailed Description

Represents a GeoJSON feature object, specifically designed to handle the parsing of alert data from external sources such as the National Weather Service (NWS).

This class facilitates the extraction of alert-related data encapsulated within a GeoJSON structure.

This class is annotated with JsonIgnoreProperties to ignore any unknown JSON properties, ensuring compatibility and resilience against changes in the JSON data structure from external APIs.

Attributes:

  • id
    - The unique identifier of the alert, typically provided by the external data source.
  • properties
    - An instance of FeatureProperties, which holds detailed information about the alert.

Methods:

  • toString() - Provides a string representation of the feature, useful for logging and debugging.
  • getFeatureAsAlert() - Converts the feature into an Alert entity, mapping relevant properties to the corresponding fields.

Usage: Instances of this class are typically created during the deserialization process of GeoJSON data. After creation, they can be transformed into Alert entities for further processing, storage, or display within the application.
