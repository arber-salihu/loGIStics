Detailed Description

Represents a geographic boundary within the system, which is stored as a multi-polygon geometry.

This class includes serialization and deserialization functionalities to handle GeoJSON multi-polygon geometry using Jackson and JTS (Java Topology Suite).

The

Boundary

class is annotated as an entity with the

@Entity

annotation, meaning it is mapped to a database table named "boundary". Each boundary has an ID, a description, and a name, along with associated

UgcZone

and

SnowUser

through many-to-many relationships.

Attributes:

  • id
    - The primary key of the boundary entity.
  • theGeom
    - The multi-polygon geometry of the boundary, handled by custom serializers and deserializers to integrate with the JSON format.
  • description
    - An optional description of the boundary.
  • name
    - An optional name of the boundary.
  • ugcZones
    - A set of
    UgcZone
    entities associated with this boundary.
  • snowUsers
    - A set of
    SnowUser
    entities associated with this boundary.

The multi-polygon geometry is mapped to the database with specific spatial data definitions, ensuring compatibility with spatial databases.
